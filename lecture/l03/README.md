### the `for` statement
Structure:
```c
for ( expr1 ; expr2 ; expr3 ) {
	statement(s)
}
```
- expr1: initialize variable
- expr2: controlling expression
- expr3: operation to perform at the end of each loop

### the `while` statement
Structure: 
```c
while (condition) {
    statements; 
}
```

### the `do-while` statement
Structure: 
```c
do {
    statements;
} while (condition);
```
